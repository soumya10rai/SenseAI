# 🧠 SenseAI – The Multimodal Assistive AI Companion

**SenseAI** is an AI-powered accessibility tool designed to empower differently-abled individuals by offering real-time support tailored to visual, hearing, and cognitive needs. Built using computer vision, speech processing, and NLP, SenseAI bridges the gap between users and digital content.

---

## 🚀 Features

- 🔍 **Visual-to-Audio Assistance**  
  - Object detection using YOLOv8  
  - OCR-based text reading (Tesseract)  
  - Converts visuals into speech for blind/low-vision users  

- 🎤 **Speech-to-Text with Emotion Detection**  
  - Transcribes speech using Whisper  
  - Detects emotions via Hugging Face sentiment analysis  
  - Chat-style output for hearing-impaired users  

- 🧘‍♀️ **Cognitive Mode (Prototype)**  
  - Simplifies complex text using NLP  
  - Designed for neurodivergent users  

---

## 🧠 Tech Stack

- Python  
- OpenCV, YOLOv8, Tesseract  
- Whisper (OpenAI), HuggingFace Transformers  
- Streamlit (for UI)
