# 🧠 SENSE-AI: Making the Web Inclusive ♿

**SENSE-AI** is an AI-powered Chrome Extension that makes any website accessible and personalized for users with disabilities — whether visual, auditory, or cognitive. It empowers users to modify web content in real-time using voice, speech-to-text, and smart UI controls.

---

## 🔍 Problem

💡 Over **1.3 billion people** globally live with disabilities, but **98% of websites** fail to meet basic accessibility standards.  
This creates a **digital divide**, excluding millions from online education, jobs, banking, and daily browsing.

---

## 🚀 Solution

**SENSE-AI** bridges that gap by using **AI** and a browser-based UI to enhance the web experience for people with:

- 👁️‍🗨️ Visual impairments  
- 🧏 Deaf or hard-of-hearing  
- 🧠 ADHD / dyslexia  
- 🔆 Sensory sensitivities

---

## 💻 Tech Stack

| Layer           | Technology Used                            |
|------------------|---------------------------------------------|
| **Frontend**     | React.js, Tailwind CSS, HTML, CSS, JS       |
| **Chrome Ext**   | Manifest v3, Content Scripts, Service Worker|
| **Backend**      | Node.js, Express.js                         |
| **Database**     | Firebase (Firestore)                        |
| **AI Features**  | Google Web Speech API, OpenAI Whisper       |

---

## 🧩 Key Features

| Feature                  | Description                                                                 |
|--------------------------|-----------------------------------------------------------------------------|
| 🎙️ Voice Navigation      | For visually impaired users — real-time control via spoken commands         |
| 📝 Speech-to-Text        | Real-time transcription of spoken content using OpenAI Whisper              |
| 🧠 Cognitive Adjustments | ADHD/dyslexia support — layout simplification, font/spacing controls        |
| 🎨 Visual Customization  | Contrast toggles, brightness, font size for color blindness/sensory needs  |
| 🔄 Persistent Settings   | User preferences saved securely using Firebase                              |
